# Koa-blog

## 项目说明：
**本项目实现前后端分离， 采用 Restful api 的规范，权限认证采用 token 来实现**

> **前端项目：** (点击此处 →)[https://github.com/liandao0815/React-blog]

## 技术栈：
> Node.js + Koa2 + MongoDB

## 项目启动:

```bash
# 安装依赖
yarn install 或者 npm install

# 启动项目
yarn start 或者 npm run start
```

## 实现功能：
- [x] 用户登录
- [x] 用户注册
- [x] 修改资料
- [x] 上传头像
- [x] 收藏文章
- [x] 发布文章
- [x] 删除文章
- [x] 更新文章
- [x] 模糊查询
- [x] 分页查询
- [x] 阅读统计
- [x] 用户评论
- [x] 文章分类
...
